# Zeratta Landing Page

Landing page do projeto Zeratta.

## 📋 Sobre

Este é o repositório da landing page oficial do Zeratta, responsável pela apresentação do projeto e capture de leads.

## 🚀 Como começar

```bash
# Clone o monorepo
git clone https://github.com/paulozeratta/zeratta-eco.git
cd zeratta-eco/packages/lp

# Instale as dependências
npm install

# Inicie o servidor de desenvolvimento
npm run dev
```

## 📁 Estrutura

```
src/
├── components/
├── pages/
├── styles/
└── assets/
```
